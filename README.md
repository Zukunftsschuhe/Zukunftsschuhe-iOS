# Zukunftsschuhe-iOS

Zukunftsschuhe: Ein Schuh der per Vibrationen den Weg navigiert, per LEDs beleuchtet und an die tägliche Bewegung erinnert. Das ist das Repository für die App des Schuhs.

# Ziele

- Schlankes Design.
- Weiterentwicklung der App.
